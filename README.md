# Ecommerce Shipment Project

This repository contains the CSE422 project for analyzing e-commerce shipping data.

## Project Contents
- **EcommerceShipment.ipynb**: Jupyter notebook with all code, analysis, and visualizations.
- **E-commerce Shipping Dataset - E-commerce Shipping Dataset.csv**: The dataset used for analysis.
- **422_project_report_.pdf**: The final project report (see for detailed findings and explanations).

## How to Use
1. Clone the repository.
2. Open `EcommerceShipment.ipynb` in Jupyter Notebook or VS Code.
3. Ensure you have Python and required libraries installed (see notebook for details).
4. Run the notebook to reproduce the analysis and results.

## Project Report
The full project report is included as `422_project_report_.pdf` in the repository.
